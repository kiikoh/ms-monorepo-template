# Microservice Monorepo Template

## Technologies

- Turborepo
- Typescript / Prettier / Eslint
- Express
- ts-rest
- and more to come...

## Getting Started

### Install

Install turborepo & pnpm globally - not technically required but highly recommended
```bash
npm i -g @turborepo/cli pnpm 
```

Clone the repo
```bash
git clone
```

Install dependencies
```bash
pnpm i
```

### Run in Development

```bash
pnpm dev
```
